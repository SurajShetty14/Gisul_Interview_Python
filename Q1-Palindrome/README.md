# Python Q1: Palindrome Sentence Checker

## Objective
Create a function that checks if a sentence is a palindrome (reads the same forward and backward, ignoring spaces, punctuation, and case).

## Problem Statement
Write a function `is_palindrome(sentence)` that:
- Takes a string as input
- Ignores spaces, punctuation, and case
- Returns `True` if the sentence is a palindrome, `False` otherwise

## Example
**Input:** `'A man a plan a canal Panama'`  
**Output:** `True`

**Input:** `'Hello World'`  
**Output:** `False`

## Instructions
1. Complete the `is_palindrome()` function in `main.py`
2. The function should:
   - Remove spaces and convert to lowercase
   - Check if the cleaned string reads the same forward and backward
3. Run your code using the "Run Code" button
4. Pass all tests in `test_lab.py` to complete this challenge

## Test Cases
Your solution will be tested with:
- Single word palindromes
- Sentence palindromes with spaces
- Non-palindrome strings
- Empty strings and edge cases
