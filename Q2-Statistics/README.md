# Python Q2: List Statistics Calculator

## Objective
Create a function that calculates basic statistics from a list of numbers.

## Problem Statement
Write a function `calculate_statistics(numbers)` that:
- Takes a list of integers as input
- Returns a dictionary containing sum, average, max, and min values

## Example
**Input:** `[12, 5, 8, 19, 3]`  
**Output:**
```
Sum: 47
Average: 9.4
Max: 19
Min: 3
```

## Instructions
1. Complete the `calculate_statistics()` function in `main.py`
2. The function should return a dictionary with keys:
   - `'sum'`: Total sum of all numbers
   - `'average'`: Mean of the numbers (rounded to 1 decimal)
   - `'max'`: Maximum value
   - `'min'`: Minimum value
3. Run your code using the "Run Code" button
4. Pass all tests in `test_lab.py` to complete this challenge

## Test Cases
Your solution will be tested with:
- Positive integers
- Negative integers
- Mixed positive and negative numbers
- Single element lists
